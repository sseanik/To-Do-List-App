# To Do List App

## Live Demo: http://128.199.248.221:5000/

**Frontend**: ReactJS + Material UI

**Backend**: Flask + SQLite3

**Functionality**:

* Add a task
* Edit a task
* Remove a task
* Search for a task

**Build**:

Database: ```sqlite3 database.db < schema.sql```

Docker: ```docker-compose up --build```
